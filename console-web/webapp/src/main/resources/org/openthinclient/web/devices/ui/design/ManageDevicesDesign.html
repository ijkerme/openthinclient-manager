Device Management Console The openthinclient Manager is a central management tool, which enables you to administrate your ThinClients via web browser. After you have defined applications and parameters, the manager allows you to easily assign them to users, locations, clients or groups thereof. Third party software can be integrated. Hardware specifications of clients can be set in the Manager. Furthermore, the Manager provides a user management, in which different groups can be combined, thus facilitating a clear and efficient administration.
